# iNotebook Web App
 Introducing iNotebook, a note-taking application with a focus on user convenience. Seamlessly sign up or log in to safeguard your notes and access them securely across devices. iNotebook goes beyond basic note creation, offering intuitive management features such as categorization, tags, and collaborative options. Enjoy rich text formatting to customize your notes, set reminders for important tasks.
